## 👨‍🚀 Getting Started

Install dependencies:

```bash
yarn install
```

---

<br>

## 🛠 Build

### To build all apps , run the following command:

```
cd frontendmentor
yarn run build
```

---

<br>

## ✍️ Develop

### To develop all apps, run the following command:

```
cd frontendmentor
yarn run dev
```

### to filter by app

#### Examples :

```
yarn dev:qr
```

```
yarn dev:chart
```

```
yarn dev:product
```
